# Aetos-Community
Rajawali adalah simbol kekuatan, ketajaman visi, ketekunan, dan kemampuan untuk terbang tinggi .AETOS membimbing anak-anak muda untuk mengasah diri, mengeksplorasi potensi, tumbuh dalam iman, mengoptimalkan pelayanan, dan akhirnya siap terbang tinggi menjadi terang dan garam dunia. "Aetos" (Αετός) berasal dari bahasa Yunani yang berarti "Rajawali"
